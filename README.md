# Esoft ASE Course Work

## Tech Stack

- Frontend: Next.js (React)
- Backend: Node.js + Express.js
- Database: MySQL

## Basic Requiremnts

- Node.js download => https://nodejs.org/en/download


- pnpm - package manager
```
npm install -g pnpm@latest-10
```

## Local Setup

### Clone the application
```
git clone https://github.com/VirajMadhu/esoft-ase-course-work.git
```

### Go to application
```
cd esoft-ase-course-work
```

### Frontend Setup

- Go to frontend folder
```
cd frontend
```

- Install dependencies
```
pnpm i
```

- Run application
```
pnpm dev
```

- App will run => http://localhost:3000/


### Backend Setup

- Go to backend folder
```
cd backend
```

- Install dependencies
```
pnpm i
```

- Run application
```
pnpm dev
```

- App will run => http://localhost:4000/
